1) 백테스트의 indicator와 비슷함.

다만 백테스트는 해당 날짜에 해당하는 데이터를 디비에서 뽑아왔지만,

트레이딩은 현재시점을 기준으로 필요한 기간만큼 디비에서 '갯수'로 가지고 옴.


백테스트 : 2018년 8월 1일 ~ 8월 25일 -> 총 25개의 데이터
트레이딩 : 2018년 8월 25일(현재날짜)부터 25개의 데이터.


2) 트레이딩에서 OHLC데이터를 가지고 오는데 조금 다른 점

예시)

30분 간격 거래를 설정
9시 12분에 거래를 시작 -> 9시 42분에 매매 시그널 생성

데이터가 6개가 필요하다고 가정.

7시 ~ 7시 30분
7시 30분 ~ 7시
8시 ~ 8시 30분
8시 30분 ~ 9시
9시 ~ 9시 30분

은 OHLC테이블에 이미 존재.

30분부터 42분까지 데이터를 30분OHLC는 아니지만 실시간으로 적용하기 위하여

OHLC테이블이 아닌 체결데이터 테이블에서 직접 OHLC를 만들어 냄.
